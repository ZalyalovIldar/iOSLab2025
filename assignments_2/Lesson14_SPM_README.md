# 📘 Lesson 14 — Swift Package Manager

## 🎯 Цель
Научиться работать со сторонними библиотеками и создавать собственные Swift Packages.

Темы урока:

- Swift Package Manager
- добавление библиотек
- управление версиями
- структура `Package.swift`

---

# 📱 Задание — Работа с библиотеками

Добавьте стороннюю библиотеку через SPM.

Пример библиотек:

- Kingfisher
- Alamofire

Используйте библиотеку в приложении.

---

## 📌 Требования

Добавить пакет через:

```
File → Add Package Dependencies
```

Использовать библиотеку для:

- загрузки изображений
- кэширования изображений

---

# 📦 Создание собственного пакета

Создать Swift Package:

```
UIComponents
```

Добавить компоненты:

- `UserCardView`
- `LoadingView`
- `AsyncImageView`

---

## 💻 Уровни выполнения

### Base

- подключена сторонняя библиотека
- библиотека используется в приложении

### Plus

- создан собственный Swift Package
- пакет подключён к приложению

### Pro

- пакет содержит несколько UI компонентов
- компоненты используются в приложении

---

## ✅ Критерии приёмки

- библиотека подключена через SPM
- пакет успешно собирается
- компоненты пакета используются в проекте
